
Programul a fost implementat folosind structura de date Hash Table. Aceasta structura consta in realizarea unui vector de liste.
In vector, pe pozitia n se gaseste lista cu elemente de tip cuvant-vector de fisiere, pentru care hash-ul cuvantului este n.
Pentru gasirea unui cuvant din dictionar se genereaza hash-ul acestuia si se cauta in lista de pe pozitia indicata de hash.
